
Archivo: pages/_document.jsx

Resumen del archivo: Este archivo contiene la configuración y etiquetas meta necesarias para el correcto funcionamiento y optimización de la aplicación NextJS llamada Rebel Tickets.

Importaciones principales: Se importan los componentes básicos de un documento NextJS como `Html`, `Head`, `Main` y `NextScript`.

Funciones principales: No hay funciones en este archivo pues se trata únicamente de una plantilla para el documento con etiquetas meta y enlaces a recursos externos.

Componentes principales: El componente principal es `<Html>` que envuelve todos los demás elementos y atributos del documento HTML. Dentro de este componente, se encuentra `<Head>` donde se incluyen todas las etiquetas meta requeridas por diferentes motores de búsqueda y redes sociales. Además, se incluye aquí el componente `<Main/>` que representa el contenido principal de la página y `<NextScript/>` que permite insertar scripts en el documento.

Este archivo forma parte de la estructura básica de una aplicación NextJS y su objetivo es configurar el documento HTML primario de la aplicación para garantizar un correcto rendimiento y optimización de los recursos utilizados.
