**Rebel** es una plataforma de tickets y eventos, desarrollada utilizando el framework NextJS y tecnologías Amazon como Amplify, DataStore y S3 entre otras. La documentación se enfoca en proporcionar una visión clara del proyecto, sus tecnologías empleadas y la estructura de archivos.

## Technologías Utilizadas

- **NextJS:** Framework de desarrollo web utilizado para crear aplicaciones React.
- **AWS Amplify:** Conjunto de herramientas y servicios que facilitan el desarrollo de aplicaciones utilizando tecnología AWS.
- **AWS DataStore:** Servicio NoSQL administrado que permite almacenar y sincronizar datos en tiempo real.
- **AWS S3:** Almacenamiento de objetos escalable y duradero.
